"오픈클로 좋다는 건 알겠는데, 매달 얼마나 나오는 거예요?"

이 질문을 정말 많이 받습니다. 유튜브에서는 멋진 데모만 보여주지, 실제 비용은 잘 안 알려주거든요. 오늘 솔직하게 정리해 드립니다.

## 결론부터: 월 5만원이면 충분합니다

최적화 없이 쓰면 월 30~60만원까지 나올 수 있습니다. 하지만 제대로 설정하면 **월 5~6만원**이면 충분합니다.

| 구성 | 월 비용 | 누구에게 맞나? |
|------|---------|---------------|
| API만 (최소) | 5,000~15,000원 | 가벼운 사용 |
| 코딩 구독 + API | 55,000~70,000원 | 일반 사용 (추천) |
| VPS 상시 운영 | 18,000~25,000원 | 24시간 돌려야 할 때 |
| 최적화 안 한 경우 | 40~80만원 | ⚠️ 이렇게 되면 안 됩니다 |

## 비용이 폭발하는 3가지 이유

### 1. 비싼 AI 모델을 기본값으로 쓰는 경우
Claude Opus 같은 최고급 모델을 모든 작업에 쓰면 금방 쿼터가 소진됩니다. "이메일 확인해"라는 단순한 작업에 최고급 모델을 쓸 이유가 없습니다.

**해결:** 단순 작업은 저렴한 모델(GPT-5 Nano, Gemini Flash), 복잡한 작업만 좋은 모델을 쓰도록 설정합니다.

### 2. 동시성 제한을 안 걸어둔 경우
에이전트가 한 작업에서 실패하면 자동으로 재시도합니다. 제한이 없으면 재시도가 50번, 100번 반복되면서 비용이 폭발합니다. 주말 사이에 20만원 이상 나온 사례도 있습니다.

**해결:** 동시 작업 수를 4~8개로 제한합니다.

### 3. 메모리 설정을 안 한 경우
오픈클로가 이전 대화를 "까먹으면" 같은 질문을 반복하면서 토큰을 낭비합니다.

**해결:** 메모리 검색 + 자동 압축을 설정하면 토큰 낭비가 줄어들고, 비용도 줄어듭니다.

## 딸깍AI는 이 설정을 어떻게 하나요?

저희가 설치할 때 기본적으로 적용하는 것들입니다:
- 작업 종류별 모델 자동 분류 (단순 → 저렴, 복잡 → 고급)
- 동시성 제한 설정 (비용 폭탄 방지)
- 메모리 최적화 (까먹지 않으면서 비용 절감)
- 보안 설정 (API 키 보호, 외부 접근 차단)

"설치만 해드리는 게 아니라, 돈 안 새도록 설정해 드립니다."

Mac Mini를 사야 한다는 얘기가 많은데, 먼저 기존 컴퓨터나 저렴한 클라우드 서버($5/월)에서 사용해 보세요. 워크플로우를 파악한 후에 필요하면 그때 사도 늦지 않습니다.
