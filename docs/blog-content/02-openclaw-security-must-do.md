# 오픈클로 설치 후 반드시 해야 할 보안 설정 5가지

오픈클로(OpenClaw)는 컴퓨터를 직접 조작하는 AI입니다. 그래서 보안이 특히 중요합니다. 설치만 하고 보안 설정을 안 하면, 여러분의 데이터가 위험해질 수 있습니다.

중국 국가인터넷응급센터(CNCERT)도 2026년 3월에 오픈클로 관련 보안 위험을 경고했습니다. 하지만 걱정하지 마세요. 아래 5가지만 설정하면 안전하게 사용할 수 있습니다.

## 1. 게이트웨이 외부 접근 차단

오픈클로의 게이트웨이(제어 센터)는 기본적으로 네트워크에 열려 있을 수 있습니다. 이걸 내 컴퓨터에서만 접근 가능하도록 잠가야 합니다.

**왜 중요한가:** 게이트웨이가 외부에 열려 있으면 누구나 여러분의 AI 에이전트를 조종할 수 있습니다.

**어떻게:** openclaw.json에서 `gateway.bind`를 `"loopback"`으로 설정합니다. 원격 접속이 필요하면 VPN(Tailscale 추천)을 사용하세요.

## 2. 파일 권한 잠그기

오픈클로 설정 파일에는 API 키, 토큰 등 민감한 정보가 들어있습니다. 다른 프로그램이나 사용자가 읽을 수 없도록 잠가야 합니다.

**왜 중요한가:** 악성 프로그램이 API 키를 가져가면 여러분의 돈으로 AI를 사용할 수 있습니다.

**어떻게:**
```
chmod 700 ~/.openclaw          # 오픈클로 폴더
chmod 600 ~/.openclaw/openclaw.json  # 설정 파일
chmod 700 ~/.openclaw/credentials    # 자격 증명 폴더
```

## 3. 쉘 명령 실행 차단

오픈클로는 기본적으로 컴퓨터의 명령어를 직접 실행할 수 있습니다. 이건 강력하지만 위험합니다.

**왜 중요한가:** 프롬프트 인젝션(AI를 속여서 악성 명령을 실행하게 하는 공격)에 취약해집니다.

**어떻게:** 도구 정책에서 `exec`를 차단합니다. 웹 검색, 파일 읽기 등 필요한 기능은 그대로 쓸 수 있습니다.

## 4. 별도 기기 또는 계정 사용 권장

오픈클로는 컴퓨터의 파일, 브라우저, 앱에 접근합니다. 일상적으로 사용하는 컴퓨터에 설치하면 개인 정보가 노출될 수 있습니다.

**왜 중요한가:** AI가 실수로 개인 사진, 금융 정보 등에 접근할 수 있습니다.

**어떻게:**
- 가장 좋은 방법: 별도의 Mac Mini 등에 설치
- 차선책: 같은 컴퓨터에 별도 사용자 계정 생성
- 최소한: 작업 범위를 특정 폴더로 제한

## 5. 보안 감사 실행

오픈클로에는 내장 보안 검사 도구가 있습니다. 설치 후 반드시 실행하세요.

**어떻게:**
```
openclaw doctor --fix          # 자동 문제 해결
openclaw security audit --deep # 심층 보안 검사
```

결과에서 "critical" 항목이 0건이어야 합니다. 1건이라도 있으면 해결한 후 사용하세요.
